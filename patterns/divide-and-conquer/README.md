# Divide & Conquer Pattern

The Divide & Conquer pattern works by breaking
a problem into smaller parts and solving them efficiently.

It is commonly used with binary search,
where the search space is repeatedly divided in half.

---

## Problems

### 1️⃣ count-zeroes
Given a sorted binary array (1s followed by 0s),
return the total number of zeroes using binary search.

### 2️⃣ sorted-frequency
Given a sorted array and a target value,
return how many times the target appears.
If the target does not exist, return -1.
